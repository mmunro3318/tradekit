# RESCOPE — 30,000-ft view + fastest path to actionable (2026-07-19)

## Where the project stands (one screen)

**TradeKit** is an event-sourced, thesis-driven trading toolkit whose core
safety property is that every action between an agent and money passes a
policy gate (R-rules, ledgered verdicts). Live trading stays structurally
locked behind four independent locks; the near-term venue is **Kraken Prop
Starter Eval 1** (balance $5k, MDL $150/day, MDD $300 static, target $500),
executed **manually by Mike** because Kraken Prop has no API and the
desktop app has no accessibility tree (UIA probe GRADE C, evidence
artifact committed).

What exists and is green (890+ tests, gate = pytest+ruff+mypy):

**The binding constraint is no longer code — it's getting one compliant
trade through the funnel before the 7-day prop inactivity close.**

## Critical path (in order; nothing else jumps this queue)

1. **Ship round** — merge `feature/bridge-read` and `feature/hud-orderbook`
   to main (tk-ship; both green and pushed). Removes two long-lived
   branches and makes `tk hud` the mainline tool. ~1 short session.
2. **First-trade rehearsal (highest value)** — dry-run the full manual
   loop end-to-end with Mike present: draft a real thesis → policy verdict
   → `tk hud` ticket (real thesis_id replaces interim provenance — small
   T6) → Mike transcribes into Kraken → report fill → `record_manual_fill`
   → reconcile. The rehearsal will flush out any policy-context wiring
   gaps (prop:* PolicyContext, R-017/R-018) that unit tests can't.
3. **Setup-gate reality check** — the pinned battery (macd_bullish +
   volume_spike 1.5, regime-gated, 4h) graded everything `wait` today.
   Correct behavior, but verify over a few days of runs that it CAN fire
   on the greenlist; if it's structurally silent, tune via doctrine
   (pullback-continuation composite is the best-evidenced family), never
   by loosening thresholds ad hoc. If no setup fires before day ~5 of the
   inactivity window, CTO adjudicates a minimal manual-thesis trade —
   still through the full funnel, policy-gated, smallest compliant size.
4. **Inactivity clock discipline** — one small executed bracket resets it.
   Repeat cadence: `tk hud` daily (Mike or scheduled), act only on
   buy/sell grades.
